除錯過程中發掘新功能的嘗試，令我樂此不疲；從程式碼到實體功能運作的轉換過程尤其讓我著迷，也深刻體會到模組化設計對於複雜系統的重要性。

「嚴謹的驗證流程，才能造就卓越的 IC 設計」——因此，我期盼能投入專業實務操作，從工作任務中持續訓練，確實理解底層邏輯功能，並掌握高效除錯的驗證方法。

我也計畫強化自己在邏輯構成與驗證方法整合的能力，讓自己對 Design House 前端至後端流程的理解更臻深入。目前積極申請台灣半導體研究中心推行的晶片實作教育訓練課程（TSRI晶片課程）。

因為有使用 Python 和 Quartus 的經驗，我正在熟悉如何整合不同語言環境以達成更好的驗證結果。同時，了解低功耗設計（如 Dynamic Voltage Scaling）的應用趨勢，也希望在實務中學習如何平衡效能與功耗，滿足不同裝置需求。

截至目前，我已修習邏輯設計與實驗、數位系統雛型設計、超大型積體電路設計導論、Python 程式設計、智慧物聯網軟硬體整合實作等課程，並具備電路學、電子學等必要基礎知識。我曾使用 Python 開發樹莓派遠端家電控制系統，也應用 Verilog RTL 設計紅綠燈數位電路系統。
